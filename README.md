# Connection 00 · Ingeniería de Prompts

Píldora DOC ROI para aprender a programar la IA generativa con una lógica clara, aplicada y gamificada.

## Objetivo

Esta Connection 00 introduce la ingeniería de prompts como una competencia práctica para trabajar mejor con IA generativa. La experiencia combina:

- Formación ejecutiva sobre prompts y variables.
- Infografías de apoyo descargables.
- Fórmulas de prompt y variables esenciales.
- Gamificación con prompts.
- Reto Cuarentena.
- Memory Jogger / Botiquín de prompts.
- Customer Care con preguntas frecuentes.
- Cierre metodológico DIIIP y KAI·ROI.

## Archivos principales

- `index.html`: entrada para Vercel. Carga el bloque publicado desde `/public/connection_00_ingenieria_de_prompts_pildora_wordpress.txt`.
- `public/connection_00_ingenieria_de_prompts_pildora_wordpress.txt`: HTML/CSS/JS principal de la experiencia.

## Publicación

El repositorio está preparado para desplegar en Vercel desde la rama `main`.
